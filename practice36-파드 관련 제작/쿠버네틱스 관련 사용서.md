## 쿠버네티스 명령어 : `kubectl`

```bash
kubectl api-resources
```

```bash
kubectl --help
```

```bash
kubectl logs --help
```

- 파드 만들고 실행시키는 명령어

```bash
kubectl run web --image=nginx:1.28 --port 80
```

- 파드를 외부와 연결 명령어

```bash
kubectl port-forward pod/web 80:80
```

## 파드는 보통 컨테이너 1개당 1개씩 생성, 노드는 각 컴퓨터 하나씩이라 생각하면된다. 그래서 파드와 노드는 연관관계가 있다. deployment를 통해 app1개를 파드 여러개로 나누고 노드를 나눠 분배하는 기능을 하는 조정한다.

- pod 정보 보기
```bash
kubectl get pods -o wide
```

- pod 접속 하기
```bash
kubectl exec -it web -- /bin/bash
```

-  nginx 'curl' 명령어로 확인 하기
```bash
curl localhost
```

- yaml 파일로 쓸 내용 찾을때
```bash
kubectl get pod web(만들어진 이름) -o yaml
```

- 매니페스트 파일을 이용하여 pod 생성하기 (.yaml 파일 이용해서 올릴때 쓰는 용어)
 ```bash
kubectl apply -f web-pod.yaml
 ```

- 그 후 파드를 외부와 연결 명령어

```bash
kubectl port-forward pod/web 80:80
```


- 쿠버네틱스 pod 종료 및 삭제
```bash
kubectl delete all --all
```

- python의 경우 yaml파일 만들고 
```bash
kubectl apply -f fastapi-pod.yaml
```
하면 에러이미지풀 오류뜸 -당연한 오류다.  => imagePullPolicy: IfNotPresent 추가 해줘야한다.
